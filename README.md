# MERN Trello clone

## Requirements

* **nodejs >= 16.14.0**
* **npm 8.3.1**
* **yarn 1.22.17**

### babel installed globally
* **npm install -g @babel/node**
* **npm install -g @babel/cli**
* **npm install -g @babel/core**

Clone project
```
$ git clone https://this_git_url <project_dir>
$ cd <project_dir>
$ npm install or yarn install 
$ npm start or yarn start
```
